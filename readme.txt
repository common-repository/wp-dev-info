Displays information on both front and back ends during development, such as memory usage, number of database queries and more.

== Description ==

View basic information about the current page you're working on (front-end), such as how many database queries were executed, how much PHP memory is being consumed and more. An admin dashboard widget is also included.


== Installation ==

Install, activate, done.

== Uninstall ==

Deactivate the plugin, delete if desired.

== Official Web Site (and support) ==

<a href="http://www.thatwebguyblog.com/post/wp-developer-info-for-wordpress/">That Web Guy Blog</a>

Hit WP Dev Info in the left nav to enable for the front-end and change settings.
